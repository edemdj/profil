# profil
MON CV
Profil – Mon CV en ligne

Bienvenue sur mon portfolio en ligne développé avec React et TypeScript. Ce projet présente mon parcours, mes compétences, et offre un moyen simple de me contacter.

Aperçu

Ce site web “MON CV” met en avant :

Mes informations personnelles et professionnelles
Un formulaire de contact fonctionnel
Mes liens vers GitHub et LinkedIn
Une interface moderne et responsive
Fonctionnalités principales

Section Contact :
Permet de m’envoyer un message directement via un formulaire ou d’accéder à mes profils GitHub/LinkedIn.
Design responsive :
S’adapte à tous les écrans (desktop, tablette, mobile).
Technologies modernes :
Utilisation de React, TypeScript et TailwindCSS pour un code propre et performant.
Technologies utilisées

React
TypeScript
TailwindCSS
Lucide React (pour les icônes)
Démarrer le projet

bash
# Clone ce repo
git clone https://github.com/edemdj/profil.git
cd profil

# Installe les dépendances
npm install

# Lance l’application en développement
npm start
Le site sera accessible sur http://localhost:3000.
